# 按功能整理 txmonad 源码文件

## 整理规范（参照 StackSet.hs 模式）

每个文件统一遵循以下结构：
1. **LANGUAGE 扩展** → **Haddock 模块注释** → **module 声明（带分组导出列表，使用 `-- * 分组名`）**
2. **导入区域**：内部模块 → 标准库 → 第三方库
3. **代码区域**：用 `-- =====...=====` 分隔注释划分大区块，每区块有中文标题
4. 每个区块内：类型定义 → 导出函数 → 内部辅助函数
5. 每个导出函数/类型必须有 `-- |` Haddock 注释
6. 内部辅助函数统一放在文件末尾的"内部辅助函数（不导出）"区域
